Predictive Analysis for US Commercial Flights Delays

It is my solo capstone project in the Data Science Bootcamp. My goal was to predict whether a particular future flight will be delayed.

I used US Commercial flights data, and completed EDA analysis using various Python packages such as Pandas, Seaborn, and Maltplotlib. Then I applied various classificaion algorithms to calculate the feature importance of dataset.

Additionally, I applied NLP techniques to calculate and visualise the feature importance of data in object data type.


Technologies

  • Pandas
  • Numpy
  • Matplotlib
  • Seaborn
  • Sciki-Learn
  • Logistics Regression
  • Random Forest
