EDA Data Analysis

I used a education dataset contains SAT scores by schools and a civic dataset contains the drug usage per state.

I used several Python packages such as Pandas, Seaborn and Maltplotlib to complete EDA Analysis and visualisation.

Additionally, I applied Hyperthesis testing to research the co-relations among use of two different type of drugs.


Technologies

  • Pandas
  • Numpy
  • Matplotlib
  • Seaborn
  • Sciki-Learn
  • BeautifulSoup
  • TF/IDF Model
  • ELI5
